###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

});

```

- B: 2
- C: 3
- D: 4

<p>

#### Answer: C

<i>Here, declared a constant variable and it is an array.Using forEach function that iterate nums array under this function when num is true (not equal to 0) then increase count by 1.In this iteration we get true for (1,2,3) so that the count increasing 3 times that's why the answer is 3.</i>

</p>
</details>
